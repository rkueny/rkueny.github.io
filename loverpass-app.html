Description du projet

L'OVER PASS vous permet d'avoir des avantages dans de nombreux bars, restaurants, loisirs et commerces de Toulouse. L'OVER PASS est sans conteste la carte de réductions à posséder si vous êtes à Toulouse !

Retrouvez facilement tout les partenaires, toutes leurs offres et leur localisation. Vous n'avez désormais plus aucune excuse pour passer à côté de L'OVER PASS !

Fonctionnalitées

  • Application mobile disponible sur iOS et Android
  • Synchronisation via une API basée sur un site Wordpress
  • Géolocalisation des partenaires
  • Mail de contact
  • Ouverture de l'application Facebook
  • Effet visuel sur le menu
